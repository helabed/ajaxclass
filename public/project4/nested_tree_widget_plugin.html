Nested Tree List Widget Plugin

Query UI widget for creating themable tree view lists or hierachical menus with the ability to nest list items by adding droppable and draggable functionality. It extends treeList widget by using jQuery widget inheritance. This widget is intended to be a very lightweight class for nesting list items by dragging child objects.
